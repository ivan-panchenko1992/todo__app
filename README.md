# TODO-APP

## About: 
implementation of classical TODO-app

## Technologies used
1. ReactJS
2. React Hooks
3. ClassNames library
3. JavaScript
4. HTML5
5. CSS  

## Instructions
1. **Fork** the repo.
2. **Clone** the forked one.
3. Run `npm install` (or just `npm i`).
4. Run `npm start` to start the project locally on host `http://localhost:3000`
